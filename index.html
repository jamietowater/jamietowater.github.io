AtomRC Dolphin FPV Wing

Long-range FPV flying wing build with iNav autonomous flight capabilities

+
FPV iNav Crossfire GPS

Overview

AtomRC Dolphin configured as a long-range autonomous FPV platform. iNav flight controller handles waypoint missions, return-to-home, and position hold. TBS Crossfire for extended range control and telemetry.

Key Features

  • iNav 7.x with GPS position hold and autonomous missions
  • TBS Crossfire RX for 10km+ range
  • Custom OSD layout with telemetry overlays
  • Tuned PID loops for smooth flight in wind
  • Return-to-home failsafe tested at 3km+ distance

Technical Details

Flight controller runs iNav with custom mixer for wing configuration. GPS module provides position data for autonomous flight modes. Crossfire telemetry feeds back battery voltage, RSSI, GPS coordinates, and flight mode to ground station. OSD configured to display critical flight data on FPV feed.

ESP32 Crossfire Ground Station

Real-time telemetry receiver and display for Crossfire RC systems

+
ESP32 Crossfire CRSF Protocol OLED

Overview

Custom-built ground station using ESP32 to receive and decode Crossfire (CRSF) telemetry data. Displays real-time aircraft position, battery status, RSSI, and flight metrics on an OLED screen. Acts as a backup telemetry monitor during long-range flights.

Key Features

  • Decodes CRSF telemetry packets (GPS, voltage, current, RSSI)
  • 128x64 OLED display with multi-screen layout
  • Data logging to SD card for post-flight analysis
  • Low-voltage buzzer alerts
  • Powered by 18650 battery with USB-C charging

Technical Details

ESP32 UART receives CRSF protocol frames at 420kbaud. Custom parser extracts telemetry fields and updates display at 10Hz. Data is timestamped and written to CSV on SD card. Low-power mode extends battery life to 8+ hours of continuous monitoring.

WS2812 LED Animation System

Custom addressable LED controller with animation engine and wireless control

+
ESP32 WS2812 FastLED WiFi

Overview

ESP32-based controller for WS2812B addressable LED strips. Features a library of custom animations, web-based control interface, and integration with Home Assistant. Drives up to 500 LEDs with smooth color transitions and effects.

Key Features

  • FastLED library for hardware-accelerated rendering
  • 20+ custom animation patterns (fire, rainbow, chase, breathe, etc.)
  • Web UI for real-time control and pattern selection
  • MQTT integration with Home Assistant
  • Audio-reactive mode using analog microphone input

Technical Details

ESP32 drives LED data at 800kHz using RMT peripheral for precise timing. Animation engine supports layered effects with alpha blending. Web server runs on port 80 with responsive UI. MQTT publishes state changes and receives commands from home automation system.

Ender 3 V2 Klipper Upgrade

Complete conversion to Klipper firmware with custom macros and tuning

+
Klipper Raspberry Pi 3D Printing Macros

Overview

Ender 3 V2 upgraded from stock Marlin firmware to Klipper running on Raspberry Pi 4. Significant improvements in print quality, speed, and features. Input shaping eliminates ringing artifacts. Pressure advance improves corner quality.

Key Features

  • Input shaping tuned with accelerometer (ringing eliminated)
  • Pressure advance calibrated per filament type
  • Custom start/end G-code macros
  • Automated bed mesh leveling before each print
  • Mainsail web interface for remote monitoring

Technical Details

Klipper offloads motion planning to Raspberry Pi, allowing 32-bit processing for smoother motion. ADXL345 accelerometer measured resonances at 42Hz (X) and 38Hz (Y). Input shaping applied with MZV algorithm. Pressure advance set to 0.055 for PLA, 0.065 for PETG. Macro library handles filament loading, purging, and calibration routines.

Home Assistant & Monitoring

Self-hosted home automation with custom integrations and monitoring dashboards

+
Home Assistant Docker MQTT Grafana

Overview

Home Assistant running in Docker on dedicated server. Integrates smart home devices, custom ESP32 sensors, and monitoring tools. Grafana dashboards track system metrics, network performance, and environmental data.

Key Features

  • MQTT broker for ESP32 device communication
  • Custom sensors: temperature, humidity, power monitoring
  • Automated lighting scenes and schedules
  • Network monitoring with Grafana + InfluxDB
  • Backup automation with daily snapshots

Technical Details

Home Assistant Core running in Docker with Mosquitto MQTT broker. ESP32 devices publish sensor data to MQTT topics. Node-RED handles complex automation flows. Grafana visualizes time-series data from InfluxDB. All services reverse-proxied through Nginx with SSL.
